Description:
Create block(s) to allow Prev/Next navigation between nodes within a nodequeue while viewing the node itself

Usage:
1. Enable module
2. Visit admin/structure/nodequeue and click the newly added "Nodequeue Pager tab" and select how many nodequeue pager blocks to make available to the site
3. On the blocks page assign a block to a region and then confugre the block to select which nodequeue to work from
